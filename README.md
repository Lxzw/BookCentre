BookCentre

1. 环境配置

  1.1 jdk7.0.25 + eclipse + mysql5.6 + tomcat7.0
  
  1.2 spring mvc 4.0.4 +  hibernate 4.3.5 + jquery 1.11
  
  1.3 git + maven 3.0 + junit 4.10
  
2. 系统部署

3. 业务设计
    
    3.1 主要需求
    
        3.1.1 用户能够通过关键字搜索想要下载的电子书

        3.1.3 管理员能够对系统内图书进行crud操作
